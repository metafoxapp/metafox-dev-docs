Installation

Checkout source code

Copy content from docker-compose.postgres.yml then to new file docker-compose.yml.

# Start docker.
docker-compose up -d


# Go into docker container.
docker-compose exec -it metafox-postgres-php-fpm bash


# Install all the dependencies using composer
composer install


# Install MetaFox (required, do not use default laravel installation.)
php artisan metafox:install


# Seed data phpFox.
php artisan phpfox:seed --force


# Run Laravel Queue in background:
nohup php artisan queue:work --daemon

Directory Structure

app/            : Laravel default, don't change
   bootstrap/      : Laravel default, don't change
   config/         : Laravel default, don't change
   database/       : Laravel default, don't change
   modules/        : All packages
       metafox/  : Core package
       platform/   : Platform Directory
   public/         : Laravel defaults
       .htaccess
       robot.txt
       index.php   : Entry point for all requests
    resources/     : Laravel default
    routes/        : Laravel default
    storage/       : Laravel default
    tests/         : Laravel default
    vendor/        : Laravel default

modules containts all packages, organize as bellow convention.

vendor_name should be your company name to avoid duplicated with others.

modules/
    [company]/
        package-1/
        package-2/

example

modules/
    metafox/
