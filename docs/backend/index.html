Docker

The fastest way to develop the platform is via docker-compose.

Checkout source code

# Start docker-compose container
docker-compose up -d


# Go into docker container.
bash ./scripts/install-via-docker.sh

The server is running at "http://localhost"

Standalone

In this case, follow step by step to install the platform.

System requirements

  • php >= 7.4
  • ext-curl
  • ext-dom
  • ext-json
  • ext-pdo
  • ext-zip
  • sodium
  • pcov
  • php-redis
  • php-imgclick
  • php-memcached
  • php-pgsql
  • php-mysql
  • php-ffmpeg

Fill Environment

Copy .env.example to .env

Fill out .env and change environment variables with property values.

Install 

./scripts/install.sh
