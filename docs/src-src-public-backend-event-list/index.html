Model Events

These are events that were called in EloquentModelObserver: modules/platform/src/Support/EloquentModelObserver.php

Model Creating Event

  • Event name: models.notify.creating
  • Example:
/**
     * @var Model $model
     */
    app('events')->dispatch('models.notify.creating', [$model]);
