step 1. Register captcha rules

```php
<?php class PackageSettingListener
{
    public function getCaptchaRules(): array
    {
        return ['create_blog'];
    }

    // other methods
  }
```

step 2. Add catpcha field to form which need captcha challenge

```php
<?php
class CreateBlogForm
{
    public function getCaptchaRules(): array
    {
        $basic = $this->addBasic();
        $basic->addField(
          Builder::captcha()->actionName('blog.create_blog')
        );
    }

    // other methods
  }
```
