## System Requirements

- node 16+

If you are using Linux/MacOS, You should install node via [Node Version Manager](https://github.com/nvm-sh/nvm).

Helpful commands

```bash
# Check current default node version
node --version

# Check nvm is installed
nvm --version

## install missing nvm
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash

# Install specify node version
nvm install 16.14.0

# Set 12.18.0 as default node excutable.
nvm alias default 16.14.0
```

## Start Dev Server

```shell
# to workbox dir
cd workbox

# install yarn
npm install -g yarn

# install dependencies
yarn && yarn bootstrap

# start dev server
yarn start

# start dev server for admin site
yarn start:admin
```

By default, deverlopment server will start at `http://localhost:3000`

## Directory Structure

Workbox contains all libraries, tools and extensions of phpfox projects. We use `lerna` to organize many packages into single repository. Let's look overall to top level of directory strucrure.

```text
workbox/
  app/
  packages/
    apps/
    framework/
    yourcompany/
  tests/
    jest/
    tools
  package.json
```

**packages/framework/**

This directory contains metafox core libraries.

**packages/apps/**

It contains modules, themes, and other extensions.

**packages/app**

It contains configuration.

Look deeper to `app/` directory

```text
app/
  dist/
  public/
    favicon.ico
    index.html
    logo192.png
    logo512.png
  src/
    index.tsx
  .env
  settings.json
  package.json
```

## Apps

**dist/**

It contains bundled react app result to deployment.

**public/**

It contains static resource to build react app.

**src/**

It contains source code of `frontend` app.

It contains application configurations.
