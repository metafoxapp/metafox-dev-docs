Metafox included the most popular react form lib `formik` and validation with `yup`,
wrapped theme in FormBuilder in order help clients build, configure flexible logic from admin panel whitout modify source code.

Our form mechanism is based on formik + material-ui, and build a form json schema for api control form field & structural. Let's take a look at this example.

## Tutorial

## Schema

```tsx
import { Form } from '@metafox/form';

const ExampleLoginForm = () => {
  const loginFormSchema = {
    container: true,
    component: 'form',
    elements: {
      email: {
        label: 'Email',
        component: 'text',
        type: 'email',
        variant: 'outlined'
      },
      password: {
        label: 'Password',
        component: 'Email',
        type: 'password',
        variant: 'outlined'
      },
      buttons: {
        container: true,
        component: 'html',
        elements: {
          submit: {
            label: 'Login',
            type: 'submit'
          }
        }
      }
    }
  };
  const initialValues = {
    email: 'jack@phpfox.com',
    password: ''
  };

  return <Form schema={loginFormSchema} initialValues={initialValues} />;
};
```

### Text

Defination

```tsx
export type TextFieldProps = {
  name: string;
  component: 'text';
  label: string;
  placeholder: string;
  readOnly?: boolean;
  disabled?: boolean;
  required?: boolean;
  maxLength?: number;
  variant?: 'outlined' | 'filled';
  type?: 'text' | 'email' | 'password' | 'number' | 'date' | 'time';
};
```

Example

```tsx
const Field = {
  name: 'title',
  component: 'text',
  label: 'Title',
  placeholder: 'Enter an title for this blog',
  readOnly: false, // optional
  disabled: false, // optional
  required: false, // optional
  maxLength: 250, // optional
  autoComplete: true, //optional,
  variant: 'outlined'
};
```
