## Introduce

In this section we create new package `note` of company `company`. The Notes app will allow users to post notes and sharing with others users.

- Allows user sharing note, including attachments, privacy.
- Configure settings, includes title, description, tags, category, etc.
- Configure permissions, admin assigns permission based on user roles.
- View notes of user's friends on thier activity streams.
- Geting notifications when others comment, like user notes.

## Create New App

**Backend**

For Backend, you can go to **AdminCP > Installed > Apps** or (use direct URL `/admincp/apps`)

Press "Create New App" button. Then, fill info of company and app name to generate new app skeleton.

In app options, press "Code Generator", and fill "note" in others to generate item type `note` skeleton.

The Skeleton for Backend of Notes app will be generated as below:

[Directory structure](backend/package#structure)

```text
packages/
 company/
    example/
      routes/
        api.php
      resources/
        lang/: define supported languages
          drivers.php: define drivers
          en/
            phrase.php: define message translation
            validation.php: define message phrases
          menu/
            items.php: define menu items
            menus.php: define menus
      src/
        Database/
          Factories/
          Migrations/
          Seeders/
        Http/
         Controllers/
         Requests/
         Resources/
        Listeners/
        Models/
        Obsevers/
        Policies/
        Providers/
        Repositories/
        tests/
          Unit/: Unit test source root
          Features/: Feature test source root
```

**Frontend**

To create skeleton for frontend, you can open Terminal, go to the source folder of ./frontend and run command:

```bash
yarn metafox create-app @company/note
```

You will see that the default skeleton for Notes app will be generated as below:

```text
packages/
  company/
    note/
      package.json
      tsconfig.json
      types.ts: define typings
      index.tsx: general export
      module.d.ts: integrate typing
      components/: define components
      pages/: define pages
        HomePage/
          Page.tsx
          layouts.json
```

## Add New Schema

Access `/admincp`, Visit `Apps` -> `Installed`, Press `Code Generator` -> `Migration`. Fill schema name etc: "notes", A new migration file is created under ./src/Database/Migrations,
file name format contain date time prefix, etc:`2022_05_23_101328_migrate_notes_table.php`.

The date time prefix help framework migration scripts known execution priority.

If you are not familiar with laravel migration, visit [Migrations](/backend/eloquent#migrations).

## Add New Model

### Backend

Access `/admincp`, Visit `Apps` -> `Installed`, Press `Code Generator` -> `Model`. Fill your package name and previous schemas, etc "notes".

MetaFox generates somes classes based on what features you have choose on the previous form.

** Has Repository? **

Generate files for repository pattern associate with the model. the pattern based on [l5-repository](https://github.com/andersao/l5-repository)

** Has Model Factory? **

Generate files for [Model Factory](https://laravel.com/docs/9.x/database-testing#defining-model-factories).

** Has Authorization ? **

Add permissions pivot for the model based on [laravel-permission](https://spatie.be/docs/laravel-permission/v5/introduction)

** Has Text Data? **

Separate text to the seconds schemas, it's helpful to reduce size of main schema.

** Has Category Data **

Create a pivot model associate with main schema to storage relation between main schema and a category schema.

** Has Tags Data **

Create a pivot schema associate with main schema to storage relation between main schema and a tags data.

** Has Activity Feed? **

Create a pivot schema to publish each note item to activity stream.

** Has Model Observer? **

Create [Observer](/backend/eloquent#observers) to listen modified of main schema.

> Check source base of generated code, auto generated code create skeletons to save your time only.
